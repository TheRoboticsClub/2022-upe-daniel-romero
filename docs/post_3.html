Week III (March 15th - 21st)

March 20, 2021 by Manuel Pérez

This week I continued producing commits for the front-end Unibotics webpage. The commits started to get more relevant, like a tutorial for the users. I had to learn some Bootstrap as I had never worked with it.

I also studied the use of Docker more in depth as one of the future goals will be to improve the speed of the RADI container using GPU.

A colleague explained me the procedures to deploy the application on the test machine so I can test the new changes before we implement it on production. As I told on previous posts, this application already has real users so it is crucial that any changes we make don’t affect the already working functionalities.

Additionally, I also set up this Github pages so there’s a registry of my work available.
