RADI 2.4.1 is finally out for all the exercises! As stated in the previous post, it increases performance on linux + nvidia hosts. The only requirement is to install nvidia proprietary drivers in order to properly utilize the GPU.

The main task of this week was to check that all the exercises were working correctly with the new RADI, as the manager and the Dockerfile went through important changes. The main use case scenarios (obtained from Unibotics dashboard) are Windows 10, Linux and to a lesser extent, Mac.

The only exercises which gave us some trouble was the ones using Drones, and also trying to run the RADI on machines with low computational power. Because of that, we are planning to create a more robust connection of the RADI with the browser, checking that every step is completed successfully before starting the next one.
