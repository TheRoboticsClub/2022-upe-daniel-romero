We are now migrating the whole Robotics Academy infrastructure to run on ROS Noetic instead of ROS Melodic. After helping with the migration, I ran some tests and published the new version, on Robotics Academy and Unibotics.

I worked with a colleague to implement a measure value (RTF) into all the current exercises. This value is helpful as it states the correlation between the time in the real world and the time of the simulation. A value of 1 indicates a perfect simulation, and values closer to 0 indicates worse performance.

I also worked on a new script that checks periodically that the Unibotics server is up and running.
