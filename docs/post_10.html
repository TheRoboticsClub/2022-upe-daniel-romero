On this week I started working on an important issue for Robotics Academy and Unibotics. Currently, the RADI image is only using CPU power. However, most of the users probably have unused computational power on their GPUs. Using this power would be very helpful as many users report low FPS and low RTF values, meaning less realistic simulations. There are other colleagues working on this task, as containerized GPU usage with ROS is not easy to achieve and there is not much documentation about it.

Additionaly, I extended the use of VNC based console on Robotics Academy for the following exercises: obstacle avoidance, drone cat mouse, follow turtlebot, color filter, vacuum cleaner and vacuum cleaner loc.
