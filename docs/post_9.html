On this week I was given the task to include a Django server into the Robotics Academy Docker Image to further simplify the use of Robotics Academy. The RADI now launches a webserver which serves the html files along with the backend. It also allows us developers to reuse pieces of code, not possible with plain html. This will increase the speed of the development as the majority of elements of each exercise page are common.

In addition, I made a new release on Unibotics and deployed it, containing performance improvements for the exercise 3D Visual Reconstruction. I also extended the user defined functions scope problems fix mentioned in the previous post to all the available exercises.
