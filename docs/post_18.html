Week XVIII (June 28th - July 4th)

July 4th, 2021 by Manuel Pérez

This is my last week as an internship in JdeRobot. I want to thank my mentors JoseMaría, Francisco and David as they helped me a lot during this months. I learnt a lot from this internship: new technologies (Docker, Django), new languages (Python), and most important, I experienced working and developing on a real and active platform.

I spent this week working on the Windows accelerated version of Robotics Academy. The solution implemented for Ubuntu hosts is not applicable to Windows as it uses VirtualGL along the Direct Rendering Infrastructure device, which is not present in Windows Subsystem Linux.

We also released a new version of the RADI which included all the exercises migrated to ROS Noetic. The whole infrastructure is now working with this new version of ROS and Python 3.

The new objectives of the platform is to continue evolving the backend (adding GPU Windows support, making it more robust, migrating it to ROS Foxy) and continue expanding the contents (adding more exercises and tools as RViz). I am thrilled to see what the future will look for Unibotics and Robotics Academy.

See you!
