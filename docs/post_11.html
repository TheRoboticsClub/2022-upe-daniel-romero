RADI 2.3.18 is out! This version includes the Django webserver I worked on. It also includes David Valladares' refactorization of the exercise pages using Django templates. This will help us developers maintain and update the code, and it also improves the user exeperience as now the users only have to launch the RADI and connect to 127.0.0.0:8000 on a browser to use Robotics Academy.

Regarding the hardware acceleration for the RADI image, I was able to achieve some performance improvements working with Jovian-Dsouza RADI fork. With this new image we are getting RTF (+0.5) and FPS (+10) improvements and lower CPU usage (-10% on each core). This image is promising, but there are still some minor issues to fix and some tests to do before deploying it as the current RADI image.
